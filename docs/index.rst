Crop row crop-row-connector
=================================

*crop-row-connector* is a python package for connecting crop rows in agricultural field data. 
It provides efficient algorithms implemented in Rust for high performance, along with a user-friendly Python interface.

*crop-row-connector* uses the output from `Crop Row Detector <https://github.com/henrikmidtiby/CropRowDetector>`_ as input and connects the detected crop rows into continuous lines.

Installation
============
*crop-row-connector* is a combined rust and python package. 
It can be installed by following the instructions in the :ref:`installation <installation>` section.
